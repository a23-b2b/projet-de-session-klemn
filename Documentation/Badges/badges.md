# Badges

Klemn offre un système de badges similaire à celui de Discord, nous utilisons des déplacements binaires pour déterminer les badges d'un utilisateur.

## Liste des badges disponibles
| Valeur | Valeur (int) | Titre     |
| ------ | ------------ | --------- |
| 1 << 0 | 1            | DevTeam   |
| 1 << 1 | 2            | Verifie   |
| 1 << 2 | 4            | BugFinder |
| 1 << 3 | 8            | DevTeam   |
